# Cloud Computing Concepts

## What is Cloud Computing?

Cloud computing is the delivery of computing services such as servers, storage, databases, networking, and software over the internet.

## Benefits of Cloud Computing

* Scalability
* High Availability
* Cost Efficiency
* Disaster Recovery
* Flexibility

## Cloud Models

### Public Cloud

Services offered over the public internet and shared across organizations.

### Private Cloud

Cloud infrastructure dedicated to a single organization.

### Hybrid Cloud

Combination of public and private cloud environments.

## Service Models

### IaaS (Infrastructure as a Service)

Provides virtualized computing resources over the internet.

Examples:

* Virtual Machines
* Storage
* Networking

### PaaS (Platform as a Service)

Provides a platform for developers to build and deploy applications.

Examples:

* App development environments
* Databases

### SaaS (Software as a Service)

Provides software applications over the internet.

Examples:

* Gmail
* Microsoft 365
* Google Drive
